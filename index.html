Bootstrap Extensions

Some CSS classes which extend Bootstrap.

This project is maintained by gonzalo-bulnes

Conventions

Following the Bootstrap conventions, all extension classes are named upon their base class name. In order to avoid any conflict in the future, I also prepend our custom classes names with custom.

Example: .custom-form-actions-bordered extends .form-actions and should be used as follow:

<div class="form-actions custom-form-actions-bordered"> ... </div>
